parqueadero con apartado grafico

El proyecto es un sistema de gestión para un parqueadero, desarrollado en Python
utilizando el framework Tkinter. El sistema se conecta a una base de datos SQLite3 para
realizar operaciones de CRUD (Crear, Leer, Actualizar y Eliminar) sobre los registros de
vehículos que ingresan y salen del parqueadero. Proporciona una interfaz gráfica intuitiva
para que los usuarios puedan gestionar las plazas de estacionamiento, registrar entradas y
salidas de vehículos, así como generar informes y estadísticas relevantes.

Requisitos previos

Antes de ejecutar el proyecto, asegúrate de tener instalado lo siguiente:

    Python  (versión 3.10.6)
    tkinter (versión 8.6)
    SQLite3 (versión 3.37.2)

Configuración

    Abre el archivo config_db.py y ejecutalo una vez para montar la base de datos con sus valores por defecto

Uso

    Ejecuta el script principal:

        - parking_app.py

    La interfaz gráfica del parqueadero se abrirá. Desde allí podrás realizar las operaciones CRUD necesarias.
